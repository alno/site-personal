В этой заметке я рассмотрю такую достаточно популярную тему, как использование регулярных выражений. Причем, рассматривать я буду применительно к <a href="http://blog.alno.name/cats/cpp">C++</a> с использованием библиотеки <a href="http://blog.alno.name/cats/boost">Boost</a>, которая содержит замечательные средства для их поддержки.

Используемый синтаксис реглярных выражений совпадает с широко известным PCRE (Perl-Compatible Regular Expressions).
 
Для использования библиотеки необходимо включить заголовочный файл <i>boost/regex.hpp</i>:
<r:code class="cpp">#include <boost/regex.hpp></r:code>
